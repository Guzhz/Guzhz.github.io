一、数组

1、arr.length的值

是根据下标最大那个决定的，如果中间没有元素则为空（,,,,)

2、修改length的值产生的影响

变大，没有值的为空（,,,,)
变小，相当于删除尾部元素

3、构造函数创建数组的细节

//多个参数，则创建的是这些参数的数组
var arr = new Array(10,20,30)
=> “10,20,30”

//一个参数时，创建的是开辟这么大空间的数组
var arr = new Array(10)
=> “,,,,,,,,,,,,,”

4、数组遍历arr.forEach((value,index,arr) => {})的三个参数分别是当前项的值，当前项的下标和当前遍历的对象
IE8一下不兼容

5、常用方法

  • arr.push(…x) => 向尾加元素 => 返回长度
  • arr.pop() => 向尾删元素 => 返回被删元素
  • arr.unshift(…x) => 向头加元素 => 返回长度
  • arr.shift() => 向头删元素 => 返回长度
  • arr.slice(start[,end]) => 切片，取左不取右，end不写默认为全部，可以为负值 => 不影响原数组，返回新数组
  • arr.splice(start_index, count [,…newElement]) => 两个参数就是删除，三个参数就是删除后添加，count可为0 => 返回新数组
  • arr.concat(newArr) => 拼接数组 => 不会对元数组产生影响
  • arr.join(“,”) => 参数是连接符，默认是逗号 => 不会对元数组产生影响
  • arr.reverse => 反转 => 会修改原数组
  • arr.sort() => 默认Unicode的编码排序
    //自定排序规则：根据回调函数的返回值设置
arr.sort(function(a, b) {
  //a始终是b的前一个元素，返回值大于1则交换，等于0不交换，小于1不交换
  //这里a-b为升序
  return a - b  
})

二、函数的call和apply方法

作用：修改函数的this指向

  • 两个都是函数的方法，需要通过函数对象去调用
  • 当对函数调用call()和apply()都会调用函数执行
  • 在调用call()和apply()时，可以将一个对象作为参数传递进去，那个这个参数（对象）将成为函数的this
var obj1 = {
  name: "我是obj1"
}
var obj2 = {
  name: "我是obj2"
}
function fun() {
  console.log(this.name)
}
========================
fun() => //打印undefined，因为没有全局变量name
fun.call(obj1) => //打印“我是obj1”
fun.apply(obj2) => //打印“我是obj2”

区别：

  • call()方法可以将实参在对象之后依次传递
  • apply()方法需要将实参封装到一个数组中统一传递（也就是只有两个参数位置）
    function fun(a, b) {
  console.log("a = ",a)
  console.log("b = ",b)
  console.log(this.name)
}
var obj = {
  name: "我是obj"
}
fun.call(obj, 1, 2)
fun.apply(abj, [1,2]) => 会自动解构

三、函数的另一个隐式参数：arguments

在调用函数时，浏览器每次都会传递两个隐式参数：
1、函数的上下文对象this
2、封装实参的对象arguments

  • arguments是一个类数组
    • arguments.length可获得它的长度
    • 但它不是数组（argumentsinstanceof array => false)
  • 即使不定义形参，也可用通过arguments来使用实参
    • 用法类似数组arguments[0]，arguments[1]
  • arguments存在一个属性callee，该属性指向当前正在执行的函数
    • arguments.callee == fun

四、new Date()

1、创建方式

//直接new对象
var d = new Date() => 当前时间

//指定日期格式：月/日/年(需要4位，2位的话ie可能歧义) 时:分:秒
var d = new Date("12/02/2021 11:20:22") => 指定时间

2、相关方法

d.getFullYear() => 取代getYear，这里返回正常4位的年份
d.getMonth() => 获得月份（0-11），需要+1
d.getDate() => 获得日期是几日
d.getDay() => 获得周几（0-6）
d.getHours() => 0-23
d.getMinutes() => 0-59
d.getSeconds() => 0-59
d.getMiliseconds() => 获得毫秒 0-999

//时间戳，计算机底层保存时间就是用时间戳
d.getTime() => 获得从格林威治标准时间的1970-1-1至今的毫秒数
//另一种方式直接获取时间戳
Date.now()

3、关于时间戳的细节

var d = new Date("1/1/1970 0:0:0")
var timestamp = d.getTime()
console.log(timestamp) => "-28800000"

解释：当前系统时间为中国北京时间，位于时区的东八区，刚好比格林威治时间早8小时

五、基本数据类型的包装类

  • 基本数据类型变量是无法使用属性和方法的
  • 但是用它定义属性的时候不会报错，是因为浏览器临时把变量封装成对象然后进行调用对象的方法，但是调用完之后又自动销毁了
  • 然后使用刚刚定义的属性又会自动装箱封装成一个新的对象来进行读取操作，因为是新对象，所以刚刚定义的属性不存在，所以打印是undefined
var a = 123

//不报错，自动装箱，封装对象并定义该属性，定义完了又销毁了
a.name = "hello"

//不报错，只是在读取的时候，也会自动装箱进行封装对象
//然后读取完了（undefined）了，又销毁了
console.log(a.name) => undefined

六、字符串详讲

字符串在在底层是以数组的形式保存的

  • str.charAt(index) => 指定位置的字符
  • str.charCodeAt(index) => 指定位置的字符的编码（Unicode）
  • String.fromCharCode(72) => “H”
  • str.concat(“拼接字符”) => 同+号
  • str.indexOf(“h”[,startIndex]) => 第一个h的index，没有找到则返回-1
  • str.lastIndexOf(“h”[,startIndex]) => 从后往前找
  • str.slice => 切片，取左不取右
  • str.substring => 类似切片，也是取左不取右（参数位置会自动交换为正序）
  • str.split(“x”) => 根据参数拆分数组，支持传入正则对象
  • str.toUpperCase() => 大写
  • str.toLowerCase() => 小写

七、正则表达式详讲

1、创建方式：构造函数、字面量

//最简单的正则表达式，构造函数创建正则对象
var reg = new RegExp("ab", "i/g")  //第二个参数为匹配模式，i为忽略大小写，g为全局匹配

reg.test("Abcd") => true

开发中常用的方式为，字面量的方式直接创建

var reg = /ab/i
reg.test("abc") => true

2、基本语法

  • “|”、[ ] 表示或关系
    • [abcd] == a|b|c|d
    • [A-Z] => 大写字母
    • [a-z] => 小写字母
    • [A-z] => 忽略大小写的字母
    • [0-9] => 数字
      • /a[bde]c/ => 检查字母中含有abc或者adc或者aec
  • ^ 表示非关系
    • [^abc] => 查找任何不在中括号以内的字符
  • {} 量词
    • /a{n}/ => 出现n次a
    • /ab{n}/ => 出现n次b
    • /(ab){n}/ => 出现n次ab
    • /ab{m,n}/ => b出现m到n次即可
    • /a{m,}/ => a出现m次以上
    • /abc/ => 表示0次或多次 => {0,}
    • /ab?c/ => ?表示0次或1次 => {0,1}
    • /ab+c/ => +表示至少出现一次 => {1,}
  • 匹配头尾
    • /^abc/ => 只能以a开头
    • /abc$/ => 只能以c结尾
    • /^a$/ => 这个a既是匹配头又是匹配尾 => ‘a’才是true
  • 通配符 . 点
    • 任意字符，除了换行符和结束符
    • 纯粹匹配.这个字符，需要转义，加反斜杠“\.” ==> 引申“\\”
    • 在构造函数中，字符串需要两个反斜杠才等同于一个反斜杠 => new RegExp(“\\.”)才等同于/\./
  • \w 任意字母、数字、下划线_ <==> [A-z0-9]
  • \W w的取反 => [^A-z0-9]
  • \d 任意数字 <==> [0-9]
  • \D 非数字 => [^0-9]
  • \s 空格
  • \S 除了空格
  • \b 单词边界 => 识别是不是一个单词
    • /\bchild\b/ => “hello child “

3、搭配字符串的方法使用

  • str.split(正则)
  • str.search(reg/str) => 返回第一次出现的下标，没有则返回-1
    • 如果传字符串则同理与str.indexOf()
    • 如果传正则对象，则比较灵活
    • 只能搜索一个，不能设置全局
  • str.match(正则) => 返回数组
    • 提取内容
    • 默认只会匹配第一次，匹配到就结束。可以全局匹配/g，同时可以忽略大小写/ig
  • str.replace(reg,”替换的内容”) => 可以全局替换 /g
