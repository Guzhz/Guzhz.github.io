前言：本笔记为VUE基础语法的使用。B站视频链接

一、插值的操作

1.1、Mustache语法

<div id="app">
  <h2>{{message}}，嘻嘻嘻！</h2>
  <h2>{{first + " " + end}}</h2>
  <h2>{{counter * 3}}</h2>
</div>
直接包裹变量、变量拼接、变量数学运算

1.2、v-once指令的使用

<div id="app">
  <h2>{{message}}</h2>
  <h2 v-once>{{message}}</h2> 	<!--不可以被修改-->
</div>
使用了v-once后，变量被修改，值不影响

1.3、v-html指令的使用

<h2 v-html="url"></h2>
url: '<a href="http://www.baidu.com">百度一下</a>'

1.4、v-text指令的使用

<h2 v-text="message"></h2>
不用Mustache语法包裹

1.5、v-pre指令的使用

<h2 v-pre>{{message}}</h2> <!--不解析 Previous-->
原样输出内容：{{message}} -> {{message}}

1.6、v-cloak指令的使用

<style>
    [v-cloak] {
        display: none;
    }
</style>

<div id="app" v-cloak>  <!--消除闪烁问题，斗篷-->
  <h2>{{message}}</h2>
</div>

二、动态绑定属性：v-bind

2.1、v-bind（:）

<div id="app">
  <a v-bind:href="myhref">bilibili</a>   <!--语法糖：只写“:”-->
</div>
data: {
	message: '你好啊',
	myhref: 'https://www.bilibili.com/'
}
VUE托管属性、绑定属性

2.2、v-bind动态绑定class属性（对象语法）

<div id="app">
  <h2 class="title" :class="{active: isActive, line: isLine}">{{message}}</h2>
  <h2 class="title" :class="getClasses()">{{message}}</h2>
  <button v-on:click="btnClick">按钮</button>
</div>
data: {
	 message: '你好啊',
     isActive: true,
     isLine: true
},
methods: {
	btnClick: function () {
		this.isActive = !this.isActive;
	},
	getClasses: function () {
		return {active: this.isActive, line: this.isLine}
	}
}
效果：点击按钮，切换active

2.3、v-bind动态绑定class属性（数组语法）

<div id="app">
  <h2 class="title" :class="['active','line']">{{message}}</h2>	//这里为字符串
  <h2 class="title" :class="[active,line]">{{message}}</h2>		//这里为数组变量
  <h2 class="title" :class="getClasses()">{{message}}</h2>		//通过方法获得数组变量
</div>

<script>
  const app = new Vue({
    el: "#app",
    data: {
      message: '你好啊',
      active: 'aaa',
      line: 'bbb'
    },
    methods: {
      getClasses: function () {
        return [this.active,this.line]
      }
    }
  })
</script>

2.4、v-bind动态绑定style属性（对象语法）

<div id="app">
  <h2 :style="{fontSize: finalSize + 'px',background: finalColor}">{{message}}</h2>
  <h2 :style="getStyle()">{{message}}</h2>
  <h2 :style="{'font-size': '50px'}">{{message}}</h2>
</div>

<script>
  const app = new Vue({
    el: "#app",
    data: {
      message: '你好啊',
      finalSize: 100,
      finalColor: 'red'
    },
    methods: {
      getStyle: function () {
        return {fontSize: this.finalSize + 'px',background: this.finalColor}
      }
    }
  })
</script>

2.5、v-bind动态绑定style属性（数组语法）

<div id="app">
  <h2 :style="[baseStyle]">{{message}}</h2>
</div>

<script>
  const app = new Vue({
    el: "#app",
    data: {
      message: '你好啊',
      baseStyle: {background: 'red'}
    }
  })
</script>
比较少用、这种用法相当于套娃

三、计算属性：computed（有缓存、效率高）

3.1、计算属性的基本用法

<h2>{{fullName}}</h2>
computed: {
	//计算属性，注重“属性”，命名方式不用get set，直接用就可以了
	fullName: function () {
		return this.firstName + ' ' + this.lastName
	}
}

3.2、计算属性的复杂操作

<div id="app">
  <h2>总价格：{{totalPrice}}</h2>
</div>

<script>
  const app = new Vue({
    el: "#app",
    data: {
      books: [
        {id: 110,name: 'Unix编程艺术',price: 119},
        {id: 111,name: '代码大全',price: 105},
        {id: 112,name: '计算机原理',price: 98},
        {id: 113,name: '现代操作系统',price: 97},
      ]
    },
    //有缓存，效率高
    computed: {
      totalPrice: function () {
        let result = 0;
        for (let i = 0; i<this.books.length; i++){
          result += this.books[i].price;
        }
        return result;
      }
    }
  })
</script>

3.3、计算属性的setter和getter

computed: {
	//简写,默认是get
	fullName: function () {
		console.log('fullName');
		return this.firstName +' '+ this.lastName;
	}
}

// 原理写法，一般我们用不到set，所以省略
computed: {
	fullName: function () {
		set() {
			console.log('只要值改变，就会默认调用')
		}
		get() {
			console.log('fullName');
			return this.firstName +' '+ this.lastName;
		}
	}
}

四、事件监听：v-on

4.1、v-on的基本使用：计数器

<div id="app">
  <h2>计数器：{{counter}}</h2>
  <button @click="increment">+</button>
  <button @click="decrement">-</button>
</div>

<script src="../js/vue.js"></script>
<script>
  const app = new Vue({
    el: "#app",
    data: {
      counter: 0
    },
    methods: {
      increment() {
        this.counter++;
      },
      decrement() {
        this.counter--;
      }
    }
  })
</script>

4.2、v-on的参数问题

无参：两者一样
	@click="btn1Click"
	@click="btn1Click()" 
有参：
	正常：		    @click="btn2Click(123)"
	undefined：   @click="btn2Click()"
	传进去了event：@click="btn2Click"
	指定获得event：@click="btn3Click(123,$event)"

4.3、v-on的修饰符

1：.stop的使用：阻止冒泡，点击div不冒泡button（div包裹button）
<div @click="divClick">
    aa
    <button @click.stop="btnClick">按钮</button>
</div>

2：.prevent的使用：阻止提交
<form action="http://www.baidu.com">
    <input type="submit" value="提交" @click.prevent="submitClick">
</form>

3：@keyup键帽监听：弹起键帽执行事件
<input type="text" @keyup.enter="keyUp"> //指定为enter键
4：.once：只能点击一次
<button @click.once="btn2Click">按钮2</button>

5.2、用户登录切换案例：问题（组件复用导致input值滞留）

需求：不需要复用，则加入key属性，给每个input添加key后则不会被复用
<div id="app">
  <span v-if="isUser">
    <label for="username">用户账号</label>
    <input type="text" id="username" placeholder="账号" key="username"> <!--标识，不一样则不复用-->
  </span>
  <span v-else>
    <label for="email">用户邮箱</label>
    <input type="text" id="email" placeholder="邮箱" key="email">
  </span>
    //思想：直接取反即可，不需要写方法
  <button @click="isUser = !isUser">切换类型</button>
</div>

data: {
	isUser: true
}

5.3、v-if和v-show的区别

    }

6.3、哪些数组方法是响应式

响应式：
	1、追加push()：this.letters.push('aaa');
    2、删除最后一个pop()：this.letters.pop();
    3、删除第一个shift()：this.letters.shift();
	4、在前面追加unshift()：this.letters.unshift('aaa','bbb');
  * 5、切片splice(位置、删除个数（空则删除后面所有）、插入)：
		- this.letters.splice(1,1);  		//删除第二个元素
		- this.letters.splice(1,3,'cc','dd','ee') 	//替换三个
		- this.letters.splice(1,0,"aaa",'ccc');      //二号位插入两个
	6、排序：sort()：this.letters.sort();
	7、反转：reverse()：this.letters.reverse();
非响应式：
	直接通过下标修改值：this.letters[0] = 'aaa';
解决：1、this.letters.splice(0,1,"aaa");   //替换
	 2、Vue.set(this.letters,0,'aaaaaa'); //对象、索引、修改值

七、书籍购物车案例知识点

7.1、for循环的另一种用法（js)

for (let book of this.books) {
	totalPrice += book.price * book.count;

7.2、VUE高级函数用法

需求：过滤数组的为100以内然后乘2的累加值。
const nums = [10,20,30,50,100,200,300];

//分开写法
let newNums = nums.filter(function (n) {
    return n<100;	//条件过滤
});
let newNums2 = newNums.map(function (n) {
    return n*2;		//进一步操作
});
//两个参数，第一个与上一个的返回值有关。默认从0为第一次遍历的上一个个值，
let total = newNums2.reduce(function (previousValue,n) {
    return previousValue + n;	//递归遍历
},0);

//合并写法：
let total = nums.filter(n => n<100).map(n => n*2).reduce((pre,n) => pre+n);

7.3、filters过滤器

//使用

filters: {
    showPrice(price) {
    	return "￥" + price.toFixed(2); //保留两位小数
    }

八、双向绑定：v-model

8.1、v-model的基本使用

<!--双向绑定：同步修改-->
<input type="text" v-model="message"> 
<h2>{{message}}</h2>

message: '你好啊'

8.2、v-model的原理

//v-bind和v-on的结合体：原理就是修改了input内容的值，通过v-on绑定事件修改了值，然后用v-bind显示值
<input type="text" :value="message" @input="message = $event.target.value">	//内容监听的简写

8.3、v-model结合radio类型

<!--双向绑定同一个变量，默认就互斥，可以不需要name-->
<input type="radio" id="male" value="男" v-model="sex">男
<input type="radio" id="female" value="女" v-model="sex">女
<h2>性别：{{sex}}</h2>

sex: '女'  //默认

8.4、v-model结合checkbox类型

//checkbox单选：同意协议场景，同意了则给点击
<input type="checkbox" id="agree" v-model="isAgree">同意协议
<button :disabled="!isAgree">下一步</button>

isAgree: false, //选中了则改为true
//checkbox多选
<input type="checkbox" value="唱" v-model="hobbies">唱
<input type="checkbox" value="跳" v-model="hobbies">跳
<input type="checkbox" value="rap" v-model="hobbies">rap
<input type="checkbox" value="篮球" v-model="hobbies">篮球
<h2>您的爱好是：{{hobbies}}</h2>

hobbies: [],   //选中了则存到数组中
//值绑定：使用场景为，不写死，动态获取，给定初始值
<label :for="item" v-for="item in originHobbies">
    <input type="checkbox" :value="item" :id="item" v-model="hobbies">{{item}}

originHobbies: ['唱','跳','rap','篮球','足球','排球']
