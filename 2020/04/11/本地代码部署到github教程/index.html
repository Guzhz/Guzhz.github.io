前言：注册GitHub有一段时间了，之前有按照教程操作过，不过后来一段时间不用又不熟练了。以至于后期突然使用时出现一些错误，导致本地文件一直提交不了。今天又熟悉了一次，这里做个笔记，以至于后面在忘记可以回顾一下。

搭建参考一

二、详细步骤

1.登录github，创建一个仓库，勾选上RENAME

name - description - public - Init - create

2.在需要部署的文件夹中打开Git bash，在命令行中输入git init把这个文件夹变成Git可管理的仓库

git init

3.通过git add .(注意这个”.”，是有空格的)，把文件添加到缓存区

git add .

4.可查看代码的状态

git status
红色为未添加到缓冲区
绿色为已添加到缓冲区

5.git commit -m “这里面写你的注释” 把文件提交的本地仓库

git commit -m “first”

6.连接远程github

git remote add origin git@github.com:Guzhz/test.git

7.仓库存在RENAME.md不为空，需要合并

git pull –rebase origin master

8.最后成功部署

git push origin master

9.重复部署

git push origin master
